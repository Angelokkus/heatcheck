Что делает HeatCheck

Четыре причины держать его включённым.

Статистика прямо в лобби

Уровень, ELO, винрейт, K/D, K/R, рейтинг, ADR и статистика на текущей карте — под каждым игроком обеих команд.

Form Level — уровень формы

Второй, независимый уровень 1–10 по последним 20 матчам: как игрок играет сейчас, а не по накопленному ELO. Зелёный — в форме, красный — просел.

Работает в 4 браузерах

Chrome, Firefox, Opera и Yandex. Одна сборка на Chromium и отдельная на Firefox — одинаковый функционал везде.

Приватность

Только официальный FACEIT Data API с вашим ключом. Никакой аналитики и трекеров — ничего не покидает браузер, кроме запросов к FACEIT.

Частые вопросы

Нужен ли ключ FACEIT API и как его получить?

Да. Расширение работает через официальный FACEIT Data API с вашим личным ключом (бесплатным).

  1. Откройте developers.faceit.com и войдите под аккаунтом FACEIT.
  2. Создайте приложение → раздел API Keys → создайте ключ типа Server-Side.
  3. Вставьте ключ в настройки HeatCheck (раздел «API и кэш») и нажмите «Проверить ключ».
Безопасно ли это? Какие данные собираются?

Никакие. HeatCheck не собирает и не передаёт ваши данные разработчику или третьим лицам. Нет трекеров и аналитики.

API-ключ хранится только в вашем браузере и отправляется исключительно на официальный API FACEIT (open.faceit.com) для получения публичной статистики.

Что такое Form Level?

Это второй, независимый уровень 1–10, который считается по вашим последним матчам (HLTV-подобный рейтинг + поправка на винрейт). Официальный уровень отражает накопленный ELO за всё время, а Form Level — как игрок играет прямо сейчас.

Стрелка на бейдже показывает расхождение: ▲ в форме, = по уровню, ▼ просел.

Как установить в Opera и Яндекс.Браузер?

Opera и Yandex Browser умеют ставить расширения из Chrome Web Store. Нажмите карточку своего браузера выше — она ведёт на нужную страницу. При первой установке браузер может попросить подтвердить «Установить расширение из стороннего источника» — это нормально.
