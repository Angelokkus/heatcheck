FACEIT Form Level

Расширенная статистика и «уровень формы» в комнате матча. Только официальный FACEIT Data API. Ключ хранится локально.

API-ключ

Получить: developers.faceit.com → создать app → Server-Side API key.

Встречи (🤝 / ⚔)

Обычно определяется сам. Заполни вручную, если счётчик встреч не появляется в комнате матча.

Кэш

Ответы API кэшируются в storage.local, чтобы соблюдать rate limits.

Параметры Form Level

Пороги — предварительная калибровка «на глаз». Их нужно подгонять под реальное распределение (см. README).

Границы между уровнями 1→2, 2→3, … 9→10.
